The Script enumeration identifies different writing systems. The values correspond to the names as defined in the Unicode standard. Note that new types may be added in the future. Applications should be ready to handle unknown values. This enumeration is interchangeable with glib::UnicodeScript. See Unicode Standard Annex 24: Script names</ulink>.
