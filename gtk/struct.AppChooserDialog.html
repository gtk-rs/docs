pub struct AppChooserDialog(_, _);

AppChooserDialog shows a AppChooserWidget inside a Dialog.

Note that AppChooserDialog does not have any interesting methods of its own. Instead, you should get the embedded AppChooserWidget using AppChooserDialog::get_widget and call its methods if the generic AppChooser interface is not sufficient for your needs.

To set the heading that is shown above the AppChooserWidget, use AppChooserDialog::set_heading.
