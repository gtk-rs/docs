fn begin_not_undoable_action(&self)

Begin a not undoable action on the buffer. All changes between this call and the call to UndoManager::end_not_undoable_action cannot be undone. This function should be re-entrant.

Get whether there are redo operations available.

true if there are redo operations available, false otherwise

fn end_not_undoable_action(&self)

fn redo(&self)

Perform a single redo. Calling this function when there are no redo operations available is an error. Use UndoManager::can_redo to find out if there are redo operations available.

fn undo(&self)

Perform a single undo. Calling this function when there are no undo operations available is an error. Use UndoManager::can_undo to find out if there are undo operations available.
