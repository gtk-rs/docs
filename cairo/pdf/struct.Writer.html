pub struct Writer<W: Write> { /* fields omitted */ }

A PDF surface that writes to a generic io::Write type (owning variant)

The Writer takes ownership of the write object. Once you're done using the surface, you can obtain the write object back using the finish() method.

If you would like the surface to reference the write object instead, use RefWriter.
